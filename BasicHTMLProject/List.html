List

  1. ol - order list
    2. Untuk membuat list terurut. Atribut dalam ol sebagai berikut:
    • 1 untuk 1,2,3,dst
    • A untuk A,B,C,dst
    • a untuk a,b,c,dst
    • I untuk I,II,III,dst
    • i untuk i,ii,iii,dst
  2. ul - unorder list
    3. Untuk membuat list tidak terurut. Atribut dalam ul sebagai berikut:
    • disc
    • square
    • circle
  3. Definition List
    4. Untuk membuat daftar terminologi dan daftar deskripsi
